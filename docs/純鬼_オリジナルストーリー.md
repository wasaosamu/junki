# 純鬼（じゅんき）- オリジナルストーリー

> 本作は「純鬼」プロジェクトのオリジナルストーリーです。
> 青鬼・魔女の家・ib などの名作ホラーゲームを参考にしつつ、完全オリジナルの物語として構成しています。

---

## 1. 概要

『純鬼』は、廃墟となった私立美術館に足を踏み入れた少女が、「純鬼」と呼ばれる存在に追われながら真実を暴き、脱出を目指すホラーアドベンチャーである。追跡と謎解きの緊張感に加え、**主人公と「純鬼」の関係**を軸にした心理ホラーとしての側面を持つ。

---

## 2. あらすじ

**15年前**、郊外の小さな私立美術館「瑠璃荘（るりそう）」で火災が発生し、学芸員を含む3名が死亡した。美術館は閉鎖され、建物は朽ちたまま放置されている。

主人公**アオイ**（17歳）の姉・**サキ**は、その美術館で働いていた学芸員だった。アオイは当時2歳。姉に「迎えにきて」と頼んだ電話が、サキを美術館に留まらせた——そう信じてきた。15年間、罪悪感を抱えながら生きてきたアオイは、姉の遺品を探しに、廃墟と化した瑠璃荘に一人で忍び込む。

館内は奇妙な静寂に包まれていた。ところどころに残る絵画、灰に覆われた調度品。そして——**純白の人影**。まるで蝋人形のように不自然な白さの、人型の「何か」が、ゆっくりと、しかし確実にアオイに近づいてくる。

逃げる。隠れる。謎を解く。鍵を探す。アオイは館を探索しながら、サキの残した日記の断片や、謎めいた絵画の数々に出会う。そして次第に明らかになる、**火災の真実**と、**「純鬼」の正体**——。

## 3. 登場キャラクター

### 3.1 主人公

| 名前 | 特徴・背景 |
|------|------------|
| **アオイ** | 17歳の女子高生。寡黙で冷静。姉・サキの死に深く傷ついており、15年間「自分が姉を殺した」と信じ込んできた。美術館に来た理由は「遺品を取る」という建前だが、本当は「真実を知りたい」「許しを請いたい」という願望がある。 |

### 3.2 過去の人物（回想・資料で登場）

| 名前 | 特徴・背景 |
|------|------------|
| **サキ** | アオイの姉。当時22歳の学芸員。優しく、アオイを溺愛していた。火災の犠牲者の一人。 |
| **神崎 瑠璃（かんざき るり）** | 美術館の創設者であり、画家。すでに故人。孤高の芸術家で、「純粋な美」を追求した。その作品には暗い寓意が込められている。 |
| **館長・副館長** | 火災時の責任者。資料により、何かを隠していた可能性が示唆される。 |

### 3.3 純鬼（じゅんき）

| 種類 | 外見・行動 | 正体（後述の真実に関連） |
|------|------------|--------------------------|
| **主な純鬼** | 細長い四肢、顔のない白い人型。ゆっくり漂うように移動する。触れるとゲームオーバー。時折、**サキに似た顔**が浮かび上がる。 | 火災の犠牲者たちの「残像」。館に刻まれた恐怖と未練が形になった存在。主な純鬼はサキの意識の断片——アオイを**館の外へ追い出そうとしている**。 |
| **小純鬼** | より小さく、数が多い。特定の部屋に群れる。 | 神崎瑠璃の作品に込められた「呪い」や、過去の来館者たちの残像。 |

## 4. 舞台：瑠璃荘（るりそう）

### 4.1 建物概要

- **築60年の木造・一部RC造**の和洋折衷建築
- 神崎瑠璃の私邸を改装した私立美術館として開館
- 火災後は封鎖され、一部は崩落。夜間は電力なし（懐中電灯必須）
- 庭園に池と枯山水があり、館と一体化した「作品」として設計されていた

### 4.2 主なエリア

| エリア | 概要 |
|--------|------|
| **玄関・エントランス** | 入り口は封鎖されているが、裏の窓から侵入可能。受付カウンターに古い来館者名簿が残る。 |
| **第1展示室** | 神崎瑠璃の初期作品。「純粋」をテーマにした少女の肖像画が並ぶ。不気味なほど整った顔立ち。 |
| **第2展示室** | 抽象画と立体作品。火災の被害が大きく、灰に埋もれている。 |
| **応接室・休憩室** | ソファ、ティーテーブル。サキの名札付きロッカーがある。 |
| **書庫** | 美術館の記録、神崎の日記、サキの業務日誌が残る。**真実の鍵**。 |
| **館長室** | 鍵付き。火災の原因に関する隠された資料がある。 |
| **地下室** | 作品の保管庫。火災の**発火点**とされる場所。神崎の「最後の作品」が安置されていた。 |
| **屋上・中庭** | 脱出ルート。ただし純鬼が最も活発に動くエリア。 |

## 5. 物語の核心——真実と twist

### 5.1 火災の真実

- 火災は**放火**だった。犯人は館長——神崎瑠璃の作品を横流しし、証拠隠滅のため火をつけた。
- サキはその現場を目撃し、館長に脅された。逃げようとしたが、地下室の扉を外から閉じられ、逃げ場を失った。
- アオイの電話は**関係ない**。サキはその日、すでに館長の不正を調査するために残っていた。

### 5.2 純鬼の正体（魔女の家・ib 的な twist）

**主な純鬼（サキの顔を持つもの）は、アオイを殺そうとしているのではない。**

- サキの残像は、アオイを**館から追い出そう**としている。館には「館長の罪」が刻まれており、アオイが深く関わると、彼女も館に囚われてしまう——サキはそれを恐れている。
- しかし純鬼は「言語」を失っている。伝えられるのは**追いかけること**だけ。アオイには「恐怖」にしか見えない。
- **トゥルーエンド**では、脱出直前に主な純鬼に「触れられる」。その瞬間、サキの記憶が流れ込む——火災の真相、館長の顔、「アオイを守りたかった」という想い。アオイは初めて「姉は自分を責めていない」と知る。

### 5.3 エンディング案

| エンド | 条件 | 内容 |
|--------|------|------|
| **「覚悟」**（トゥルー） | 全謎解きクリア、脱出直前に純鬼に触れる（選択肢で「逃げない」を選ぶ） | サキの記憶を見る。真相を知ったアオイは館を出て、証拠を警察に届ける決意を固める。 |
| **「脱出」**（ノーマル） | 全謎解きクリア、純鬼に触れずに脱出 | 館から逃げ切るが、真実は知らないまま。後日、アオイは「また来る」と呟く。 |
| **「囚われ」**（バッド） | 特定の選択を誤る、または時間切れ | アオイは館に取り残され、純鬼の一員となる。 |
| **「償い」**（隠し） | 館長室の資料を全て読み、地下室で特定のアイテムを使用 | 館長の亡霊（？）と対峙。アオイは姉の代わりに「裁き」を下す——といった暗喩。 |

## 6. ゲームの流れ（基本構造）

1. **侵入** - 瑠璃荘の裏口から忍び込む
2. **遭遇** - 純鬼が現れ、追跡が始まる
3. **探索** - 懐中電灯を手に、館内を探索。鍵、メモ、サキの遺品を集める
4. **謎解き** - 神崎の絵画に隠された暗号、日記の断片、金庫のパスワードなど
5. **追跡・回避** - 純鬼から逃げる、隠れる、または「正体を知った上で」触れる（トゥルー用）
6. **脱出** - 屋上または中庭から外へ

## 8. ゲームの特徴・恐怖演出

- **緊張の緩急**: 探索中の静寂と、純鬼出現時の急変。足音は少なく、**気配**で察知する演出
- **即ゲームオーバー**: 純鬼に触れられると即死。ただしトゥルー用に「意図的に触れる」選択肢を用意
- **謎解き**: 絵画、日記、数字の暗号。神崎瑠璃の「作品世界」と現実が絡む
- **心理ホラー**: 「追いかける存在」の正体が明かされることで、恐怖が悲しみに変わる
- **閉塞感**: 夜の廃墟、懐中電灯の限られた光、出口のない廊下

## 10. 純鬼プロジェクト実装メモ

- 純鬼の追跡AI：プレイヤーの「油断した瞬間」（長い謎解き後、アイテム取得直後など）に出現しやすい
- サキのペンダント：トゥルーエンド条件のフラグとして使用
- 館長の証拠：エンディング後の「後日談」や続編に繋げる余地
