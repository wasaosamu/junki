# コメント記述ルール

- **コメント開始列**: 行末コメントの `#` は **第72列** に揃える（左揃え）。
- **コードの長さ**: コメントを同じ行に書く場合、コード＋空白は **69文字以内** とする（69文字＋`  # ` で72列目に `#`）。
- **長い行**: コードが69文字を超える場合は、**コメントのみ次の行に書く**。次の行は同じインデント＋空白で `#` が72列目になるようにする。

例（同じ行に収まる場合）:
```python
        pygame.init()                                                       # 説明
```

例（長いためコメントを次行に）:
```python
        self.screen = pygame.display.set_mode((SCREEN_WIDTH, SCREEN_HEIGHT))
                                                                        # 描画先ウィンドウ。…
```
